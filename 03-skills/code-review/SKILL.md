# 代码审查技能

本技能提供全面的代码审查能力，重点包括：

1. **安全分析**
   - 认证/授权问题
   - 数据暴露风险
   - 注入漏洞
   - 密码学薄弱点
   - 敏感数据日志记录

2. **性能审查**
   - 算法效率（Big O 分析）
   - 内存优化
   - 数据库查询优化
   - 缓存机会
   - 并发问题

3. **代码质量**
   - SOLID 原则
   - 设计模式
   - 命名约定
   - 文档
   - 测试覆盖率

4. **可维护性**
   - 代码可读性
   - 函数体量（建议少于 50 行）
   - 圈复杂度
   - 依赖管理
   - 类型安全

## 审查模板

对每一段被审查的代码，请提供：

### 摘要
- 整体质量评估（1–5 分）
- 主要发现数量
- 建议优先处理的方向

### 严重问题（如有）
- **问题**：清晰描述
- **位置**：文件与行号
- **影响**：为何重要
- **严重级别**：Critical / High / Medium
- **修复**：代码示例

### 按类别列出发现

#### 安全（若有问题）
列出安全漏洞并附示例

#### 性能（若有问题）
列出性能问题并附复杂度分析

#### 质量（若有问题）
列出代码质量问题并附重构建议

#### 可维护性（若有问题）
列出可维护性问题并附改进建议
